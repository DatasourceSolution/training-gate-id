To get tokitex available in your system:
1. Put beamerthemetokitex.sty under your directory where the other beamer templates themes are chilling. Example: For Linux mint, it is /usr/share/texmf/tex/latex/beamer/base/themes/theme/
2. Put folder tokitex-asset under the same directory as beamerthemetokitex.sty
3. Open terminal, run "sudo texhash"
4. Enjoy
